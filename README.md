## 苹果为什么要锁住 MacBook Pro 的屏幕亮度？

先说清楚这件事，很多人误以为是屏幕硬件限制。

实际上，搭载 M 系列 Pro 或 Max 芯片的 MacBook Pro 14 寸和 16 寸，屏幕峰值亮度高达 1600 尼特。这个亮度在大太阳下直接工作都够用。但苹果默认只在观看 HDR 视频或使用 Metal 应用程序时才能达到这些亮度级别。日常用 macOS 桌面、写文档、改代码，系统最高只给你约 500 尼特——亮度条拉满了，也只是碰到了一道隐形的天花板。

这块屏幕本来就能那么亮。苹果只是没把权限开放给普通日常使用。
